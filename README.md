🎮 2D Dev Portfolio

An interactive developer portfolio built as a 2D game.
Instead of scrolling through a traditional website, visitors can explore a pixel-art world and interact with different objects to discover projects, skills, and information.

Inspired by the classic old-school Pokémon games, this portfolio turns the idea of browsing a portfolio into a small playable experience.

🌍 Live Concept

In this portfolio:

You control a character and move around the map

Different objects represent sections of the portfolio

Interacting with them reveals information like:

Projects

Skills

Contact details

Developer information

The goal was to create something more memorable than a traditional portfolio website.

✨ Features

🕹️ Game-style navigation instead of scrolling

🧱 Tile-based world with pixel art

💻 Interactive objects that display portfolio information

🎮 Keyboard movement controls

📱 Works in a browser like a normal website

🎨 Retro aesthetic inspired by classic RPG games

🛠️ Tech Stack

JavaScript

HTML / CSS

Node.js

Vite

Kaboom.js (2D game engine)

Tiled Map Editor (for creating maps)

Pixel art tilesets
